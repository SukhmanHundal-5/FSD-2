# Experiment 16 - Testing of Frontend and Backend

## 🎯 Aim

To test backend APIs and frontend components and integrate automated testing using GitHub Actions.

---

## 🔧 Backend

* Developed using Flask
* APIs tested using pytest
* CRUD operations verified

### Commands

cd Testing/Backend
pytest

---

## 💻 Frontend

* Developed using React (Vite)
* Components tested using vitest

### Commands

cd Testing/Frontend
npm install
npm run test

* Runs tests automatically on every push
